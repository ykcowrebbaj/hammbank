(1 to 100).toList
List.range(1, 101)
1.until(101).toList
((1 to 100) :\ List[Int]())(_ :: _)

(1 to 100).toList.sum

var res = 0
(1 to 100).toList.foreach( res += _ )

var res: Int = 0
(1 to 100).toList.foreach( (x:Int) => res += x )

3、判断题：scala中，变长参数可以放在参数列表的任意位置

错。如果方法除了变长参数外，还有其他参数，那么变长参数必须在参数列表的最后。

4、判断题：

代码 

def printMsg(first: Int, msg: String){
  val ret = msg * first
  println(ret)
  ret
}

val message = printMsg(2, "abc")

那么，message的值等于"abcabc"

错。message的值等于()，任何方法体前没有=号则该方法返回值都为()，只有在方法体前有=号时才把方法体中的最后一个语句作为该方法的返回值。
